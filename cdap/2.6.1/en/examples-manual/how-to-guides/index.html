How-To Guides¶

Designed to be completed in 15-30 minutes, these guides provide quick, hands-on instructions for building the “Hello World” of any development task with CDAP.

  • Realtime Data Processing With a Flow: Learn how to process data in realtime with CDAP. You will discover how easy it is to scale out applications on CDAP.
  • Ingesting Data into CDAP using Apache Flume: Learn how to use Flume to ingest data into CDAP in real-time.
  • Batch Data Processing With CDAP: Learn how to run MapReduce jobs to process data in CDAP.
  • Analyzing CDAP Data from BI Tools: Learn how to integrate CDAP Datasets with Business Intelligence tools to perform data analysis.
  • Storing Timeseries Data: Learn how to store timeseries data in CDAP Datasets.
  • Consuming Twitter Data in Realtime: Learn how to consume a live tweets stream in realtime, one of the common tasks of big data applications that power the social analytics.
  • Iterative Data Processing with Apache Spark: Learn how to run a Spark program to process data in CDAP.
