======================
Cask Hydrator Overview
======================

..  youtube::  JeqJrBW1UCU
   :align: center

..    :height: 100%
..    :width: 800px

Cask Hydrator is an extension of CDAP that combines a user interface with back-end services
to enable the building, deploying, and managing of data pipelines.
